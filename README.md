# Clawcut - Agentic Video Editing Skill

Clawcut adalah engine pengeditan video berbasis FFmpeg yang dirancang khusus untuk AI Agent. Terinspirasi dari Editly, namun dioptimalkan untuk performa "Sat-Set" tanpa dependensi yang berat.

## Fitur Utama
- **Declarative Editing**: Edit video menggunakan instruksi JSON.
- **FFmpeg Native Engine**: Cepat, ringan, dan stabil di berbagai environment.
- **Smart Transitions**: Mendukung filter `xfade` (wipe, fade, slide, etc).
- **Pendig Branding**: Integrasi otomatis logo dan font Poppins.

## Struktur Folder
- `/clawcut/clawcut.py`: Engine utama.
- `/clawcut/presets/`: Template pengeditan (News, Cinematic, etc).
- `/clawcut/assets/`: Bahan mentah (fonts, logos, audio).
- `/clawcut/outputs/`: Hasil render.

## Cara Penggunaan (v1.0)
Saat ini Clawcut masih dalam tahap pengembangan fitur. Gunakan `clawcut.py` untuk melakukan render berbasis JSON.
